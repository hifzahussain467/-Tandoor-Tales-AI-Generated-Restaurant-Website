# -Tandoor-Tales-AI-Generated-Restaurant-Website
This project is an AI-generated restaurant website developed as part of Assignment 01 for the Generative AI course at Superior University. The goal of this project was to explore how artificial intelligence can assist in designing and developing a complete website through prompt-based generation.

# ✨ Features
- Fully AI-generated website  
- Responsive and modern UI design  
- Restaurant menu with PKR pricing  
- Multiple structured pages  
- Clean navigation and layout  
- User-friendly experience  
 🤖 AI Tool Used
Lovable AI was used to generate the website design and structure through effective prompt engineering.

## 💡 Prompt Used
"Create a professional and modern restaurant website including Home, Menu, Gallery, About, and Contact pages. Use PKR pricing and ensure a clean, responsive, and attractive user interface."
## 📚 Learnings / Reflection
This project helped me understand the practical application of Generative AI in web development. I learned how prompt engineering plays a crucial role in getting accurate and useful outputs from AI tools.

Additionally, I gained knowledge about website structure, UI/UX basics, and how AI can significantly reduce development time. This assignment improved my creativity, problem-solving skills, and confidence in using AI for real-world projects.
